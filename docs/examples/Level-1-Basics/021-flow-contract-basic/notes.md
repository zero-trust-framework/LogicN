# 021 — flow contract basic

**Concept:** Flow Contract — `contract { types, intent, events }`

A contract block groups flow-local metadata:
- `types { }` — flow-local type aliases (not promoted to global scope)
- `intent { }` — machine-readable purpose (feeds IGO)
- `events { }` — declared event emissions

**AI rule:** Read the contract first to understand what/why, then the body for how.
