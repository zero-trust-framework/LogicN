# LogicN Compliance Security

`logicn-compliance-security` maps compliance controls to LogicN security,
network and application-kernel policy evidence.

Use this package for:

```text
required security control references
security evidence references
security exception workflow metadata
policy attestation metadata
security compliance report aggregation
```

It must not duplicate `logicn-core-security`, `logicn-core-network` or
`logicn-framework-app-kernel`.
