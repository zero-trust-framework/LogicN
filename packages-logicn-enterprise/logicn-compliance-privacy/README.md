# LogicN Compliance Privacy

`logicn-compliance-privacy` defines privacy policy and report contracts.

Use this package for:

```text
personal data classification
data minimisation
purpose limitation
consent references
lawful-basis references where applicable
data subject request workflow references
cross-border data transfer metadata
privacy-safe logs and reports
```

It must not claim that an app is legally privacy-compliant by itself. It makes
privacy-relevant facts explicit and reportable for review.
