# LogicN Web Router

`logicn-web-router` defines browser route and navigation contracts.

Use this package for:

```text
typed browser routes
route parameter validation
safe link generation
navigation event policy
route-level data fetch contracts
route preload and streaming policy
browser route reports
```

It must not become a server router, API server, CMS router or full single-page
application framework. Server API routes belong in `logicn-framework-api-server`;
browser route rendering belongs in `logicn-web-render` and app-specific code.
