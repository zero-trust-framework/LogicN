# LogicN Web Components

`logicn-web-components` defines browser component boundary contracts.

Use this package for:

```text
typed component props
component output contracts
safe child rendering
slot/content safety policy
component effect declarations
component accessibility report hooks
component render report metadata
```

It must not become a design system, component library, CSS framework or page
builder. Real visual components can be app-specific or external-framework
adapters; this package owns the LogicN contracts that keep component inputs,
effects and HTML output safe.
